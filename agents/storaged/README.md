# storaged installers (placeholder)

`storaged` is the Go agent developed in the `plusclouds/s3.agent` repo (see its `docs/storaged-design.md`). No install/packaging scripts exist yet for it beyond that repo's own `scripts/push.sh`/`deploy.sh` (SSH-based binary push for development). A real cross-platform installer, when built, belongs here, following the same pattern as `agents/vm-service`.
